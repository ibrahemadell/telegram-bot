# دليل استيراد البيانات من Google Sheets

## الخطوات المطلوبة:

### 1. الحصول على ملف credentials.json
- اذهب إلى [Google Cloud Console](https://console.cloud.google.com/)
- أنشئ مشروع جديد أو اختر مشروع موجود
- فعل Google Sheets API
- أنشئ Service Account
- حمل ملف الـ JSON وأعد تسميته إلى `credentials.json`
- ضعه في نفس مجلد المشروع

### 2. مشاركة الـ Spreadsheet
- تأكد من مشاركة الـ Spreadsheet مع الـ Service Account email
- العنوان يكون مثل: `your-service-account@your-project.iam.gserviceaccount.com`

### 3. تشغيل الاستيراد
```bash
python run_import.py
```

## البيانات التي سيتم استيرادها:
- ✅ بيانات الخزنة (الدخل والصرف)
- ✅ قائمة العملاء
- ✅ قائمة الموردين
- ✅ قائمة الموظفين مع الرواتب

## ملاحظات مهمة:
- البيانات ستُضاف إلى قاعدة البيانات الحالية
- لن يتم حذف البيانات الموجودة
- تأكد من نسخ احتياطي قبل الاستيراد
- الاستيراد آمن ولن يؤثر على البوت

## استكشاف الأخطاء:
- إذا كان هناك خطأ في التاريخ، تأكد من تنسيق التاريخ (DD/MM/YYYY أو YYYY-MM-DD)
- إذا كان هناك خطأ في الأرقام، تأكد من عدم وجود فواصل أو رموز غريبة
